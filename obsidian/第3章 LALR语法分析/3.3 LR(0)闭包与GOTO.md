# 3.3 LR(0)闭包与GOTO

#### 目标

定义 `Item` 结构体（`prod_id`/`dot_pos`，含 `operator<`/`operator==`），实现 `Closure` 和 `GotoFunc` 两个核心运算。文法已通过 `Syntax` 自动插入 `ROOT -> S`（产生式 0），无需手动增广。

#### 实现

1. `Item` 结构体
    1. 成员：`int prod_id`（产生式 ID）、`int dot_pos`（圆点位置，0 表示最左）
    2. `operator<` 先按 `prod_id` 再按 `dot_pos`；`operator==` 为 `= default`

2. `Closure(std::set<Item> items)`
    1. 按值接收（调用方 move），`queue<Item>` 工作队列驱动 BFS
    2. 弹出项 `[A -> α·Bβ]`：若 `B` 为非终结符，从 `symbol_to_productions_[IdOf(B)]` 获取 `B` 的所有产生式，插入 `{prodId, 0}`
    3. 循环至工作队列为空

3. `GotoFunc(const std::set<Item>& items, int symbolId)`
    1. 遍历项集，筛选圆点后符号编号等于 `symbolId` 的项，圆点右移一位 → `moved`
    2. 对 `moved` 调用 `Closure` 后返回

4. gtest 覆盖（`lalr_parser_lr0_test.cpp`）
    1. `ClosureFromRootStart`、`ClosureIdempotent`、`GotoOnE`、`GotoOnPlusEmpty`、`GotoOnT`、`GotoOnEofEmpty`
