# 3.6 前瞻符传播

#### 目标

为每个 `LR(0)` 状态中的每项计算精确的 `LALR(1)` 前瞻符集合。采用自发-传播不动点算法：初始项 `[ROOT -> ·S]` 获得 `{$}`，然后沿 `GOTO` 边将前瞻符传播到目标状态的对应项，同时为非终结符展开的闭包项自发计算 `FIRST(β)` 前瞻符。迭代至所有项的前瞻符不再变化。

#### 实现

1. 数据结构
    1. 定义 `using StateLookaheads = std::map<Item, SymbolSet>` 作为单个状态中各项的前瞻符表
    2. `lr0_lookaheads_[stateId][item]` 存储某状态某 `LR(0)` 项的前瞻符集合

2. 初始化
    1. `lr0_lookaheads_[0][{0, 0}]` 插入 `{$}`——增广项 `[ROOT -> ·S]` 的唯一有效上下文是 `EOF`

3. 不动点主循环（快照避免迭代中修改）
    1. 对每个状态 `s` 取当前 lookahead 快照
    2. **GOTO 传播**：对项 `[A -> α·Xβ]`（`LA`），若 `GOTO(s, X) = t`，则 `t` 中对应项 `[A -> αX·β]` 继承 `LA`
    3. **自发生成**：若 `X` 为非终结符 `B`，计算 `FIRST(β)` 作为闭包项 `[B -> ·γ]` 的自发前瞻符；若 `β` 全部可空则 `LA` 也加入自发集
    4. 将自发前瞻符合并到当前状态 `s` 中所有 `B` 的产生式闭包项 `[B -> ·γ]`

4. 编写 `gtest` 验收
    1. `ExpressionReduceLookahead`：`E -> T·` 的前瞻符 = `{$, +}`
    2. `RootItemLookahead`：`[ROOT -> ·E]` 的前瞻符 = `{$}`
    3. `ClosureSpontaneousLookahead`：`A -> B c` 中 `B` 的闭包项获 `{c}`
    4. `NullableChainLookahead`：全可空链 `A->B, B->C, C->ε` 的前瞻符正确传播 `{$}`
