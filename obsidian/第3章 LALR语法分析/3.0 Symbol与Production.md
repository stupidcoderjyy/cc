# 3.0 Symbol与Production

#### 目标

在 `src/syntax/production.h` 中定义文法的基础数据结构和工具类型，实现：`SymbolType` 三态枚举（`kTerminal`/`kNonTerminal`/`kEof`）、`Associativity` 三态枚举（`kLeft`/`kRight`/`kNonAssoc`）、`Symbol` 符号结构体（`name`/`type`/`priority`/`assoc`）及其 `operator==`/`operator<` 和 `SymbolHash` 哈希器、`UnorderedSymbolMap`/`SymbolMap` 模板别名、`SymbolSet` 有序集合别名、`Production` 产生式结构体（`id`/`head`/`body`/`priority`/`assoc`）。

#### 实现

1. 定义枚举类型
    1. `enum class SymbolType { kTerminal, kNonTerminal, kEof }`，`kEof` 用于结束标记 `"$"`
    2. `enum class Associativity { kLeft, kRight, kNonAssoc }`，分别表示左结合、右结合、非结合

2. 定义 `Symbol` 符号结构体
    1. 成员：`std::string name`、`SymbolType type`、`int priority = 0`、`Associativity assoc = kLeft`
    2. 构造：`Symbol(name, type, priority, assoc)` 四参数、`Symbol(name, type)` 双参数、默认构造
    3. `operator==` 按 `name` 和 `type` 比较；`operator<` 先按 `type` 再按 `name`

3. 实现 `SymbolHash` 和类型别名
    1. `SymbolHash` 的 `operator()` 组合 `hash<string>(name) ^ (int(type) << 1)`
    2. `template <typename V> using UnorderedSymbolMap = std::unordered_map<Symbol, V, SymbolHash>`
    3. `template <typename V> using SymbolMap = std::map<Symbol, V, std::less<>>`
    4. `using SymbolSet = std::set<Symbol, std::less<>>`

4. 定义 `Production` 产生式结构体
    1. 成员：`int id`（唯一编号）、`Symbol head`（左部）、`std::vector<Symbol> body`（右部，空表示 epsilon）、`int priority`、`Associativity assoc`
    2. 构造：`Production(id, head, body, priority, assoc)` 五参数，默认构造
