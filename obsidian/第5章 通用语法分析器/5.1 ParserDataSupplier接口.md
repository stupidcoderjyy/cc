# 5.1 ParserDataSupplier接口

#### 目标

在 `common/compile/parser.h` 中定义语法分析器的数据类型和供应器接口。

#### 实现

- `Symbol { bool is_terminal; int id }` — 文法符号，`id` 在 `terminals_`/`non_terminals_` 空间内编号
- `Production { int id; Symbol head; vector<Symbol> body }` — 产生式
- `Property`/`PropertyTerminal` — 归约时传递的数据：`PropertyTerminal` 持有 `unique_ptr<Token>`
- `ReduceFunc = function<void(const vector<unique_ptr<Property>>&)>` — 归约回调
- `ParserDataSupplier` 纯虚接口：
  - `TokenMappersCount()` — token类型到符号的映射表长度
  - `NonTerminalSymbolsCount/TerminalSymbolsCount/StatesCount/ProductionsCount()` — 维度
  - `InitReduceActions/InitGoto/InitActions/InitTokenToSymbol/InitProductions` — 初始化
- `ParserAction` 枚举：`kActionAccept=(1<<16), kActionShift=(2<<16), kActionReduce=(3<<16)`
