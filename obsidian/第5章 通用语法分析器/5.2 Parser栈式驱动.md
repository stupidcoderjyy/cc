# 5.2 Parser栈式驱动

#### 目标

实现 `Parser::Parse(Lexer& lexer, CompilerInput& ci)`——状态栈 + `Property` 栈驱动的移入-归约循环。

#### 实现

1. 构造初始化
    1. `ParserDataSupplier` 初始化：`token_to_symbol_`（token→符号映射）、`actions_`（状态×终结符）、`goto_`（状态×非终结符）、`productions_`、`reduce_actions_`

2. `Parse` 主循环
    1. `states.push_back(0)`，首个 token 从 lexer 获取（EOF直接返回）
    2. 查询 `order = actions_[state][token_to_symbol_[token->Type()]]`，解码 `type = order >> 16`，`target = order & 0xFFFF`
    3. `kActionShift`：`states.push(target)`，token转 `PropertyTerminal` 压入 properties 栈，读下一个 token
    4. `kActionReduce`：取产生式 `p = productions_[target]`，从两栈各弹出 `body.size()` 个元素 → 调 `reduce_actions_[p.head.id](body)` → `p_head = new Property()` 压入 → `states.push(goto_[state][p.head.id])`
    5. `kActionAccept`：最终归约 ROOT产生式 → `reduce_actions_[0](body)` → 返回
    6. 其他 → `OnFailed` 抛 `CompileError`
