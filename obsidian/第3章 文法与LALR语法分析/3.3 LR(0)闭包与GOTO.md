# 3.3 LR(0)闭包与GOTO

#### 目标

定义 `Item { prod_id, dot_pos }`（含 `operator<`），实现 `Closure` BFS 和 `GotoFunc` 核心运算。起始项为 `{0, 0}`（ROOT产生式圆点在最左）。

#### 实现

- `Closure`：按值接收 items，`queue<Item>` 驱动 BFS——圆点后非终结符 `B` 的全部产生式 `{p_id, 0}` 加入结果集
- `GotoFunc(items, symbolId)`：筛选圆点后符号匹配的项 → `dot_pos+1` → `Closure`
