# 3.5 LALR合并与FOLLOW集

#### 目标

`MergeLALRStates` 按项核心 `(prodId, dotPos)` 将同心 `LR(0)` 状态合并为 `LALR` 状态，重构 `GOTO`。`ComputeFollowSets` 不动点迭代计算所有非终结符的 `FOLLOW` 集。

#### 实现

- `MergeLALRStates`：`map<set<Item>, int>` 按核心分组 → 同心状态映射到同一新ID → `lr0_to_lalr_` 记录映射 → 重构 transitions，同状态同符号不同目标则抛异常"not LALR"
- `ComputeFollowSets`：`FOLLOW(ROOT) = {$}` → 迭代扫描产生式 `A -> αBβ`：将 `FIRST(β)` 加入 `FOLLOW(B)`；若 `β` 全可空，则 `FOLLOW(A)` 全加入 `FOLLOW(B)`
