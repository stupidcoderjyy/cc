# 3.4 LR(0)项集族构建

#### 目标

`BuildCanonicalCollection` 用 `BFS` 从初始项集 `Closure({0,0})` 出发构建完整 `LR(0)` 状态机，所有状态存储在 `lr0_states_` 中。

#### 实现

- 用 `map<set<Item>, int>` 做项集去重和状态索引映射
- worklist循环：弹出状态，收集所有圆点后符号 → 各自 `GotoFunc` → 新项集若未见过则创建新状态并入队 → 记录 `transitions[sym] = target`
- `LR0State`：`items`（项集）+ `transitions`（`map<int, int>` 符号ID→目标状态）

gtest：`lalr_parser_canonical_test.cpp` 验证状态数、初始态项集、转移正确性、去重
