# 3.2 FIRST集计算

#### 目标

在 `LALRBuilder` 中实现 `ComputeFirstSets` 不动点迭代，处理空产生式与可空性传播。通过 `First(symbolId)` 和 `HasEpsilon(symbolId)` 访问结果。

#### 实现

- `InitSymbols` 为全部终结符、非终结符、EOF分配整数ID；`BuildProductionIndex` 建立非终结符→产生式ID列表
- `ComputeFirstSets`：终结符的 `FIRST = {自身}`，空产生式左部 `HasEpsilon = true`
- 不动点主循环：`A -> X1...Xn`——将 `FIRST(Xi)` 注入 `FIRST(A)`，`Xi` 不可空则停止，全部可空则 `HasEpsilon(A) = true`
