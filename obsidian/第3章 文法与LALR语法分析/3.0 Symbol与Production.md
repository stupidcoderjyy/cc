# 3.0 Symbol与Production

#### 目标

在 `cc/syntax/production.h` 中定义文法的基础数据结构。`SymbolType` 三态枚举、`Associativity` 三态枚举、`Symbol` 符号结构体（`name`/`type`/`priority`/`assoc`）、`SymbolHash` 自定义哈希器、`UnorderedSymbolMap`/`SymbolMap`/`SymbolSet` 类型别名、`Production` 产生式结构体（`id`/`head`/`body`/`priority`/`assoc`）。

#### 实现

- `Symbol::operator==` 按 `name`+`type` 比较（忽略 `priority`/`assoc`），`operator<` 先 `type` 后 `name`
- `SymbolHash` 组合 `std::hash<string>(name) ^ (int(type) << 1)`
- `UnorderedSymbolMap<V>` = `unordered_map<Symbol, V, SymbolHash>`
- `SymbolMap<V>` = `map<Symbol, V, less<>>`，`SymbolSet` = `set<Symbol, less<>>`
- `Production` 五参数构造，`body` 为空表示epsilon
