# 3.1 Syntax文法存储

#### 目标

实现 `Syntax` 类：`AddProduction` 首次调用自动插入 `ROOT -> head` 作为产生式0，后续调用不重复。符号属性（优先级/结合性）支持三种设置粒度，`FindSymbol` 按 `name+type` 查找。

#### 实现

- `AddProduction`：`RegisterSymbol` 将 `head`/`body` 中符号归入 `terminals_`/`non_terminals_`；首次调用时 `InsertRootProduction` 插入 `ROOT -> head`（id=0）
- `SetSymbolProperties(sym, priority, assoc)`：根据 `sym.type` 选择 map，`try_emplace` 或原地修改
- `SetProductionPriority/Associativity`：`ranges::find_if` 按 `prodId` 查找后修改
- `FindSymbol(name, type)`：构造临时 key 查找，返回 `const Symbol*` 或 `nullptr`
- `root_symbol()` 固定返回 `{"ROOT", kNonTerminal}`，`end_symbol()` 返回 `{"$", kEof}`
