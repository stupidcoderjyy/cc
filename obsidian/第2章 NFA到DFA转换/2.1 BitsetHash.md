# 2.1 BitsetHash

#### 目标

在src/util/下创建bitset_hash.h模板文件，实现：std::bitset的自定义哈希函数模板BitsetHash（要求N为64的倍数以使用高效分块哈希），以及对应的BitsetMap类型别名模板。该工具将在后续关卡中被DFABuilder用于字符签名分组和NFA状态组映射。

#### 实现

1. 实现BitsetHash哈希模板
    1. 在src/util/bitset_hash.h中定义template <size_t N> struct BitsetHash，添加static_assert约束N > 0且N % 64 == 0
    2. 在operator()中通过检测sizeof(std::bitset<N>)和N/8的差异判断是否存在padding字节——若有padding则回退到std::hash<std::string>对to_string()结果哈希
    3. 无padding时通过std::memcpy将bitset的底层存储提取为uint64_t分块数组
    4. 对每个uint64_t块调用std::hash<uint64_t>并用boost hash combine公式（0x9e3779b9黄金比例常数、异或、左移、右移）混合所有分块哈希值

2. 提供BitsetMap类型别名
    1. 定义template <typename V, size_t N> using BitsetMap = std::unordered_map<std::bitset<N>, V, BitsetHash<N>>，方便后续代码以精简语法使用自定义哈希的bitset映射表
