# 2.1 BitsetHash

#### 目标

在 `cc/util/bitset_hash.h` 中实现 `std::bitset<N>` 的自定义哈希模板 `BitsetHash` 和对应的 `BitsetMap` 类型别名。该工具是后续DFA构建中 `NFAGroup` 映射和字符签名分组的基础。

#### 实现

1. `BitsetHash` 模板
    1. `static_assert` 约束 `N % 64 == 0` 确保高效分块哈希
    2. `operator()` 中检测 `sizeof(std::bitset<N>)` 与 `N/8` 的差异判断 padding；有 padding 则降级为 `std::hash<string>` 对 `to_string()` 哈希
    3. 无 padding 时 `memcpy` 提取 `uint64_t` 分块数组，逐块 `std::hash<uint64_t>` + boost hash combine（`0x9e3779b9` 常数、异或、移位）

2. `BitsetMap` 类型别名
    1. `template <typename V, size_t N> using BitsetMap = std::unordered_map<std::bitset<N>, V, BitsetHash<N>>`
