# 1.1 NFANode

#### 目标

定义NFA图的最基本单元——节点类。实现边类型枚举（`kNoEdge`/`kSingleEpsilon`/`kDoubleEpsilon`/`kChar`）以区分四种转移方式，提供 `AddEpsilonEdge` 的单/双参数重载实现自动类型升级（无→单→双），`AddCharEdge` 绑定 `CharPredicate` 断言函数和转移目标，`ForEachEdge` 统一遍历所有后继节点。

#### 实现

1. 定义 `NFANode` 类和 `EdgeType` 枚举
    1. 在 `cc/lex/nfa_node.h` 中定义 `enum class EdgeType : uint8_t { kNoEdge, kSingleEpsilon, kDoubleEpsilon, kChar }`
    2. 定义 `using CharPredicate = std::function<bool(int)>` 作为字符断言函数类型
    3. 节点成员：`edge_type_`、`next1_`/`next2_`（裸指针，不持有所有权）、`predicate_`、`accepted_`、`id_`（由外部管理器赋值）

2. 实现边添加逻辑
    1. `AddEpsilonEdge(NFANode* next)`：当前 `kNoEdge`→`kSingleEpsilon`；当前 `kSingleEpsilon`→`kDoubleEpsilon`
    2. `AddEpsilonEdge(next1, next2)`：直接设置为双epsilon转移
    3. `AddCharEdge(predicate, next)`：绑定断言并设置 `kChar` 类型

3. 实现 `ForEachEdge` 遍历
    1. 根据 `edge_type_` 分发，对非 `kNoEdge` 和非 `kChar` 类型调用回调 `func(this, next)`

4. 访问器和比较
    1. `id()`/`accepted()`/`edge_type()`/`next1()`/`next2()`/`predicate()` getter
    2. `set_id()`/`set_accepted()` setter
    3. `operator==` 和 `operator!=` 基于 `id_` 比较
